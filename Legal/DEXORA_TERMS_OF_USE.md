## 2. What Dexora Does

Dexora turns your iPhone into a remote interface for a coding agent running on your paired computer.

The app can send messages, attachments, voice transcription requests, git actions, worktree actions, and other commands to the bridge on that computer. The bridge then talks to your local coding agent and tools.

Dexora does not run the coding agent for you. It does not provide a hosted development workspace.
