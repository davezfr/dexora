# Dexora Public Documents

This folder keeps the public pages needed for Dexora TestFlight review.

Publish these files:

- `DEXORA_PRIVACY_POLICY.md` as `https://davezfr.github.io/dexora/privacy/`
- `DEXORA_SUPPORT.md` as `https://davezfr.github.io/dexora/support/`
- `DEXORA_TERMS_OF_USE.md` as `https://davezfr.github.io/dexora/terms/`

If upstream reference files exist in a private/source checkout, do not publish them as Dexora pages:

- `PRIVACY_POLICY.md`
- `TERMS_OF_USE.md`

Those files belong to the original Remodex project and are not Dexora's public legal documents.

Dexora's public pages should keep the same product boundary:

- small-group beta testing
- self-hosted computer-side bridge
- no Dexora account system
- no ads, tracking, RevenueCat, StoreKit, or in-app purchases in this build
- unofficial app, with open-source attribution to Remodex where needed
